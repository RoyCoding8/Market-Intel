# Frontend

Next.js 16 dashboard for the Market Intelligence Agent.

## Development

```bash
npm ci
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js 16 (standalone output)
- React 19
- Zustand (state management)
- Tailwind CSS 4
- Recharts (charts)
- Radix UI (primitives)

## Tests

```bash
npm test
```
